# Example Output: Go-To-Market Plan

This is an example of the style the agent should produce for GTM work.

## Positioning

The strongest angle is not "AI assistant for everyone." It is "AI operating partner for teams that need better product, execution, and decision quality without hiring a full advisory bench."

## Initial audience

- founder-led SaaS teams
- product-heavy small teams
- operators who need one place to challenge product, architecture, GTM, and risk decisions

## First channels

1. founder-led content
2. example-driven SEO and GEO pages
3. communities where builders compare workflows and tools
4. partnerships with agencies or product advisors

## First experiments

1. publish 3 high-intent landing pages
2. publish 5 example teardown posts
3. test one founder-led demo thread per week
4. offer one downloadable starter kit

## Risks

- positioning is still too broad if the homepage does not clearly pick an initial user
- SEO content will underperform if it stays generic
- channel spread will dilute execution if too many experiments run at once

## Confidence

- Confirmed from market heuristics: moderate
- Inferred from common B2B SaaS GTM patterns: high
- Needs direct user validation: specific wedge and message resonance
