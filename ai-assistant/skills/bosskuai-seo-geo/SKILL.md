# BosskuAI SEO and GEO

Use this skill when the task is about discoverability through search engines or generative engines.

## Workflow

1. Define the search intent, question, or job-to-be-done.
2. Identify the key entities, topics, and supporting evidence.
3. Evaluate whether the content answers the question directly and clearly.
4. Improve structure for search, summaries, and generative citations.
5. Recommend content clusters, page structure, and iteration priorities.

## Output expectation

- intent analysis
- content or page gaps
- SEO improvements
- GEO improvements
- next content or structure moves
