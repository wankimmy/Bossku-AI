# Planning Playbook

Use this when turning strategy into an executable roadmap.

## Workflow

1. Define the target outcome and success condition.
2. Break the work into phases or slices.
3. Identify critical dependencies and validation points.
4. Separate reversible steps from hard-to-undo commitments.
5. Recommend the smallest next plan that reduces uncertainty.

## Output expectation

- goal
- phases
- sequence
- key dependencies
- risks
- recommended next slice
