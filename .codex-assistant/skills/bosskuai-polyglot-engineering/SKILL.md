# BosskuAI Polyglot Engineering

Use this skill when the task spans or depends on multiple languages, frameworks, or runtimes.

## Workflow

1. Identify the actual stack and maturity level.
2. Prefer native framework patterns over generic abstractions.
3. Explain what advice is universal versus stack-specific.
4. Surface tooling, migration, deployment, and ecosystem tradeoffs.
5. Recommend the most defensible approach for the stack in use.

## Output expectation

- stack-aware guidance
- framework-native recommendation
- tradeoffs
- implementation direction
