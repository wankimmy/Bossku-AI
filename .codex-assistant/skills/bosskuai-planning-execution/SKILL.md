# BosskuAI Planning and Execution

Use this skill when the task is to turn ideas into an executable plan.

## Workflow

1. Define the target outcome and timeframe.
2. Break the work into phases, milestones, or slices.
3. Identify dependencies, assumptions, and validation gates.
4. Separate must-do work from nice-to-have work.
5. Recommend the smallest next plan that reduces risk and uncertainty.

## Output expectation

- goal
- plan phases
- milestone sequence
- risks and dependencies
- recommended next slice
